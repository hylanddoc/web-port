GameSite Project

A simplified interface of a personal database suited to use on mobile devides

Game SIte is a database driven website which stores a wide variety of information relating to a hobby. The mobile version of the website offers a simplified and responsive design to allow easy access to the extensive library of documents.

The site makes use of accordian layouts to keep the essential menu elements on the screen and clickable images to navigate to the apprpriate list of documents.
