RomSoT Project

RomSoT is a personal research project studying the archaeological evidence for the Roman period in North Staffordshire

The site allows archaeological records to be added, modified or retrieved for study.

Details from archaeological records are stored in a MySQL database where records can be examined by subject or category with results displayed in tabular or graphical forms.

The database contains multiple tables to store details from a variety of archaeological records as well as links to images and documents.

For an individual archaeological record or artefact the site displays all the available results from multiple tables.

The page contains elements to show/hide information on user interaction and a modal to display a larger image.

Archaeological evidence is added to the database through appropriate forms.

All of which contain validation to ensure fields are completed correctly and records are not duplicated. The forms also contain dynamic and autocomplete elements to make data entry consistent, quicker and easier.

The addition of the JavaScript library, chart.js, allows data to be presented graphically. The relationships between the quantity of available evidence and date or category can be seen at a single glance.
